Estos son mis comandos

  1. cat fichero-- Muestra el contenido de un fichero
  2. pwd-- Muestra el directorio actual o de trabajo
  3. history-- Muestra el listado de comandos usados por el usuario
  4. clear-- Limpia la pantalla y coloca el promot al principio de la misma
  5. mkdir-- (mkdir nom_directorio) Crea una nueva carpeta
  6. code .-- Para abrir directamente Visual Studio Code
  7. ls-- Lista los archivos y directorios dentro del archivo de trabajo
